# LYStatusBarHUD
一个简单的状态栏指示器

## 显示成功信息
...objc
[LYStatusBarHUD showSuccess:@"加载成功"];
...

## 显示失败信息
...objc
[LYStatusBarHUD showError:@"加载失败"];
...

## 显示正在加载信息
...objc
[LYStatusBarHUD loadingMessage:@"正在加载..."];
...

## 显示普通信息
...objc
[LYStatusBarHUD showMessage:@"加载"];
...

## 隐藏信息
...objc
[LYStatusBarHUD hide];
...
